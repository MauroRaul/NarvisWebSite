Termini di Servizio

Accedendo e utilizzando l'applicazione Narvis, accetti di essere vincolato dai presenti Termini di Servizio.

Narvis è un ambiente di scrittura avanzato fornito all'utente "così com'è". Ti impegni a utilizzare il servizio per scopi legali e non lesivi dei diritti di terze parti. Essendo una versione beta, potresti incorrere in malfunzionamenti occasionali. Ti invitiamo ad abilitare il salvataggio automatico ed eseguire backup regolari.

3. Servizi Integrati (Google Drive)

L'utilizzo delle funzionalità Cloud di Narvis richiede il collegamento a Google Drive. Il nostro software sfrutta le API di Google nel rispetto completo delle Norme sui dati utente dei servizi API di Google. L'accesso a tali servizi è regolamentato anche dai Termini di Google.

4. Modifiche ai Termini
